# v1.0 — Component Library (Historical)

Original hcm-ui component library implementation. Components have been moved to `apps/web/src/` and the `packages/hcm-ui` package has been removed. This file is kept as a historical record.

## What Was Built

### Phase 1: Foundation Hooks

- [x] useSelection (single/multi mode)
- [x] useFieldVisibility (localStorage persistence)
- [x] useFilter (draft/applied pattern, lifecycle callbacks: onApply, onReset, onDraftChange)

### Phase 2: Form System (Superseded)

- [x] ~~FormFieldConfig types~~ → Removed. Forms now use TanStack Form directly.
- [x] ~~FormField renderer~~ → Removed. Fields composed with `form.AppField` + `FieldWrapper` + prop getters.
- [x] ~~FormBuilder~~ → Removed. No node-based rendering.
- [x] useCrudForm → Renamed to `useCrudDialog`. Stripped of form value management. Now pure dialog lifecycle.
- [x] ~~CrudSheet~~ → Renamed to CrudDialog. Reads from props, not context.
- [x] AsyncCombobox (decoupled from FormFieldConfig, standalone component)
- [x] useDebounce hook

### Phase 3: Table System (Superseded)

- [x] ~~createColumns + ColumnConfig~~ → Removed. Use TanStack Table `ColumnDef` directly.
- [x] ~~useDataTable (custom)~~ → Removed. Use DiceUI's `useDataTable`.
- [x] ~~useRemoteTableQuery~~ → Removed. Use controlled data props.
- [x] FilterPanel compound component (kept)

### Phase 4: TreeTable + Chooser

- [x] useChooser hook
- [x] ChooserDialog component
- [x] ChooserField component
- [x] useTreeTable hook
- [x] TreeTable component

### Phase 5: Workflow + Shared

- [x] ConfirmDialog
- [x] StatusBadge
- [x] useWorkflowActions (with stateReducer + onPipelineChange)
- [x] WorkflowModalFooter
- [x] ActionInputDialog
- [x] PageHeader

### New: Form Integration Layer

- [x] TanStack Form contexts (`createFormHookContexts` + `createFormHook`)
- [x] FieldWrapper (reads field context, renders label/error chrome)
- [x] Prop getters (getInputProps, getTextareaProps, getSelectProps, getCheckboxProps, getDateProps, getChooserProps)
- [x] CrudFormBridge (creates form, syncs isDirty)

### New: DataTable (DiceUI)

- [x] Installed `@diceui/data-table` via shadcn CLI
- [x] DataTable, DataTableToolbar, DataTableSortList, DataTablePagination, DataTableColumnHeader, DataTableViewOptions
